# Hobby Time Protector Skill

## Übersicht
Dieser Skill schützt deine Hobby-Zeit, indem er automatisch Zeitblöcke in Apple Calendar für Hobbies erstellt und deinen Fortschritt trackt.

## Features
- **Automatische Kalender-Blockierung** - Plant feste Zeitblöcke für Hobbies
- **Konfigurierbare Kategorien** - Wandern, Schwimmen, Lesen, Podcasts, etc.
- **Fortschritt-Tracking** - Speichert km gelesen, Podcasts gehört, etc.
- **Automatische Erinnerungen** - Benachrichtigungen vor Hobby-Zeit

## Installation
```bash
cd /Users/melf/melflin/skills/hobby-protector
npm install
```

## Konfiguration
Siehe `config.json` für Hobby-Einstellungen.

## Usage
```bash
node index.js
```

## Tracking-Datei
Fortschritt wird in `progress.json` gespeichert.
